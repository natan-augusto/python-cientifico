# Functions in Python
</div>

A function is any block of code that can be reused to perform an operation.

# Functions:
- ```print()```: Displays something on the screen.
- ```time.sleep()```: Freezes the terminal for a certain period of time.
- ```input()```: Receives user data.

You can also create functions using ```def```:

- ```def main()```: Create a function "main"
- ```main()```: Call the function "main"
